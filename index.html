Ideas Principales.

  • Las hojas de estilo en cascada o CSS es un leguaje que define la aparinecia de un documento escrito en un lenguaje de marcado.
  • Son un mecanismo simple para agregar estilo o diseño por ejemplo, fuentes, colores espaciado a documentos web
  • os elementos de la pagina web creados con Html se les dara la apariencia que se desee utilizando Css: formas, tamaños, colores, margenes, tipo de letra, etc.

Apuntes de Clase.

Como agregar CSS al Html

Hay tres formas de poner el estilo de CSS en el HTML:

  • Hojas de estilo incrustada
  • Hoja de estilo en linea
  • Hoja de estilo extrena

Hoja de estilo incrustada

Una hoja de estilo puede incrustarse en un documento con la etiqueta "style"

La etiqueta "style" se coloca en la cabecera "head" del documento.

Una hoja de estilo incrustada puede usarse en un documento unico. Si la misma hoja de estilo se usa para multiples documentos, entones seria mas apropiada una hoja de estilo externa.

Hoja de estilo en linea

  • Puede usarse estilos en linea mediante el atributo style
  • El atributo style puede ser aplicao a cualquier elementeo HTML
  • El estilo en linea pierde muchas de las ventajas de las hojas de estlo al mezclar contenido on presentacion.

Hoja de estilo externa

  • Una hoja de estilo externa pede ser enlazada a u documento HTML mediante el elemento "link" de HTML.
  • La etiqeta "link" se coloca en la cabecera "head" del documento
  • El atributo type es ocional se usa para especificar un tipo de medio.

Resumen.

CSS es el lenguaje utilizado para describir la apariencia y el formato de un documento escrito en HTML o XML. Mientras HTML se encarga de la estructura del contenido, CSS se encarga del diseño, el estilo y el aspecto visual de una página web.
